NAME
    words - Retrive and Print words form URL

DESCRIPTION
    Usage:
        python3 words.py 'url'

FUNCTIONS
    fetch_words(url)
        Grabbing a list of words from a URL..

            url: URL of a UTF-8 doc

            A list of strings containg the words from a doc

            url: where the text be at yo!
